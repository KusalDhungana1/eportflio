# Portfolio

**Live Website:** [https://kusaldhungana1.github.io/eportflio/](https://kusaldhungana1.github.io/eportflio/)

## About
This is my portfolio showing how I build websites, covering structure, design, and interactivity.

## What it does
Instead of just reading about web development, users can actually interact with it. The website features live tools like an API simulator to show how data flows, visual sliders to demonstrate CSS layouts in real-time, a live screen-size checker for Bootstrap, and a working JavaScript console to show how code executes and handles errors dynamically.
